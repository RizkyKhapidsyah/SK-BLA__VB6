BLA - Beta 001 
------------------------
Aggiornamento 22.03.2001
------------------------


Bla è un client IRC/IRCX interamente scritto in Visual Basic 6.
Non vuole essere un'alternativa a MIRC o ad altri programmi già
esistenti ma bensì un ambiente in cui l'utente possa capire i protocolli
di trasmissione e di conseguenza costruirsi il proprio client personale.

Utilizzo
--------
Una volta aperto BLA per collegarsi è necessario caricare il file di script oppure
andare su Utilità e, dopo aver selezionato Script Editor, costruirsi il proprio client.

Dalla Spy Status i comandi vengono inviati SENZA \(slash) davanti al comando.
Dalle finestre di Query o Canale (Room) per inviare comandi che non siano dei PRIVMS
è necessario inserire la \(slash) prima del comando.

Il linguaggio di script utilizzato non è altro che Visual Basic Script.

- Consigli per l'installazione:

Bla può essere scompattato in una qualsiasi posizione dell'HD.
Se all'avvio appaiono degli errori relativi alla mancanza di OCX non preoccupatevi
sarà sufficiente copiare (o spostare) gli ocx presenti all'interno di Bla nelle
seguenti directory (o cartelle)

Windows 95/98 :  C:\windows\system
Windows 2000  :  C:\winnt\system32

Di solito non è necessario riavviare il sistema.

In caso di eventuali mancanze di librerie (dll) e altri componenti (ocx) 
o vi cercate su INTERNET la libreria necessaria oppure ci mandate una mail con il 
nome della libreria e provvederemo ad inviarvela.
